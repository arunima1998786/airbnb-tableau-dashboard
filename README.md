## Project Overview
This project presents an interactive Tableau dashboard analyzing Airbnb listings across New York City. The dashboard focuses on pricing patterns, booking demand, customer reviews, neighbourhood distribution, and host performance.

The goal of this project is to derive actionable insights related to market dynamics, customer behavior, and location-based pricing differences.

## Tools Used
- Tableau
- Data Visualization
- Data Analysis
- Dashboard Design

## Key Insights
- Manhattan and Brooklyn dominate the Airbnb market and contribute the largest share of listings and bookings
- Manhattan has the highest average prices, while Queens and Bronx are relatively more affordable
- Booking demand shows clear seasonality, with a major peak in June
- Customer reviews have grown significantly over time, indicating strong platform expansion and user engagement
- Entire home listings generally receive better reviews, while private rooms drive strong booking demand
- A small group of hosts generates significantly higher reviews, showing concentration in host performance
- Pricing varies sharply even at neighbourhood level, indicating strong micro-market differences
